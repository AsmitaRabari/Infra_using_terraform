**Terraform AWS Infrastructure Setup**

This Terraform project provisions a basic web infrastructure on AWS, including:

  A custom VPC with public subnets
  
  An Internet Gateway and Route Table for public access
  
  Security Groups allowing SSH and HTTP traffic
  
  Two EC2 instances (web servers) in separate subnets
  
  An Application Load Balancer (ALB) distributing traffic between the instances
  
  An S3 bucket for storage
